### To run the program

Build the ContractDemo.csproj and run Program.cs

Press "s" to start the application

Enter the date and contracts that fit the certain decision rules will be notified

If the date is entered and the contract is already notified, the notification won't be created

Three types of notifications exists and these can be found in inputs/config.json

Different type of contracts can be found in inputs/contracts.json

Already notified contracts can be found in inputs/notification_log.json
