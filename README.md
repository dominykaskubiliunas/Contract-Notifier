### To run the program

Build the ContractDemo.csproj and run Program.cs

Press "s" to start the application

Enter the date and contracts that fit the certain decision rules will be notified. Various decision rules can be found in inputs/config.json

If the contract has already been notified, the notification won't be created 

Different types of contracts can be found in inputs/contracts.json

Already notified contracts can be found in inputs/notification_log.json
